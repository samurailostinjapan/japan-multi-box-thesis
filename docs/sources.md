## Rules

- Use the original official release whenever available.
- Record the data period, unit, definition, and revision status.
- Separate fact, reported opinion, inference, and forecast.
- Treat surveys according to sample size and composition.
- Treat individual businesses and personal cases as qualitative evidence.
- Re-verify every current number before publication.
